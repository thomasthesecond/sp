Surgery Partners Pattern Library

Helper classes

These CSS classes can be used on any HTML element to adjust styles or layout on the fly. The source code for these classes can be found in src/assets/scss/utils.scss.

Classname Purpose
hidden Hide content visually and from screen readers; can also use the HTML5 boolean `hidden`
visuallyhidden Hide content only visually, but have it available for screen readers
invisible Hide content visually and from screen readers, but maintain layout; can be used along with `opacity` for animating content in and out
clearfix Contain floats
align-left Left align text
align-right Right align text
align-center Center align text
float-left Float content left
float-right Float content right
text-tiny Sets font size to “tiny”
text-positive Colors text green; used to signal a positive action
text-negative Colors text red; used to signal a negative action
